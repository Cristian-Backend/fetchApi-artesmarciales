Bienvenidos a mi Proyecto de Artes Marciales

Este es un proyecto que desarrollé como parte de una landing page interactiva, donde consumí una API de MongoDB para mostrar información sobre diversas artes marciales. A través de esta aplicación, se pueden explorar las características de cada arte marcial, como su origen, fundador, filosofía y más.
